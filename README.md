# 📖 Voca

- 보카 영단어장 만들기

## 💻 사용된 기술 및 라이브러리

- React
- TypeScript
- Json-server
- sweetalert2

### 🪄 DB정보 불러오는 방법

- json-server --watch ./src/db/data.json --port 3001
